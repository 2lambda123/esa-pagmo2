Schwefel¶

struct pagmo::schwefel¶

The Schwefel problem.

This is a scalable box-constrained continuous single-objective problem. The objective function is the generalised n-dimensional Schwefel function:

\[ F\left(x_1,\ldots,x_n\right) = 418.9828872724338 n - \sum_{i=1}^{n} x_i\sin \sqrt{|x_i|}, \quad x_i \in \left[ -500,500 \right]. \]
The global minimum is in \(x_i=420.9687, i = 1..n\), where \( F\left( 420.9687,\ldots,420.9687 \right) = 0 \).
